# Bank Management System

A Java-based banking application designed to demonstrate core Object-Oriented Programming (OOP) principles. The system provides basic banking functionalities, including customer management, account operations, fund transfers, and account searching.

## Features
- Create and manage customer profiles.
- Create and manage bank accounts.
- Deposit and withdraw funds.
- Transfer money between accounts.
- Search for customers by name.
- Calculate the total balance across multiple accounts.
- Identify the bank with the highest number of accounts.

## Project Structure

### Customer
Represents a bank customer and stores:
- Customer ID
- Customer Name

### BankAccount
Represents a bank account and handles:
- Account Number
- Balance Management
- Deposit Transactions
- Withdrawal Transactions
- Account Ownership Information

### Bank
Represents a bank and manages:
- Bank Information
- Customer Accounts
- Account Storage and Expansion
- Account Tracking

### Driver
Contains:
- Application Entry Point (main method)
- Sample Data Initialization
- Testing and Demonstration of System Features
